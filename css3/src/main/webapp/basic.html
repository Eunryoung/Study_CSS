CSS basic

<CSS 문법>

선택자 { 속성 : 속성값; }

<CSS 적용방식 3가지>

  • 1. External CSS : 외부 파일 방식. 스타일 소스를 웹문서와 별개로 만들어 연결하는 형태로 적용
    -> 통일된 스타일 가능, 수정/보수 용이
  • 2. Internal CSS : 내장 방식. 스타일 소스를 웹문서의 head 안에 만들어 적용
    -> 웹문서에 종속됨(해당 페이지에만 적용)
  • 3. Inline CSS : 인라인 방식. 스타일 소스를 태그 뒤에 직접 적용
    -> 소스가 길어지고 유지/보수에 불리하고 통일성 있는 사이트 구축이 어려우나, CSS 적용 우선 순위가 가장 높음
    -> 본문만 수정 가능한 개발 환경인 경우에 사용

<대표 선택자 3가지>

  • 1. tag 선택자 : 태그 그대로를 선택자로 활용. 자동 적용, 강제 적용.
  • 2. id 선택자 : id를 선택자(#id)로 활용. 중복 적용 불가. 주로 구조적 목적에 쓰임.
  • 3. class 선택자 : 클래스를 선택자(.class)로 활용. 중복 적용 가능. 조합 가능. 주로 디자인적 목적에 쓰임.
